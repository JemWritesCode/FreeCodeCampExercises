Introduction

Github is by far the most popular version of source control in use today.

The source in source control refers to the files that make up a project. Simply put source control helps you handle all the files in a project. You can:

  • Create backups
  • Organize your work
  • Make working as a team easier
  • Show off your skills
  • and much more!

    ACP
    git add

    Add does stuff

    git status

    Check the status of what files have been added

    git commit -m "what was done"

    Write down what you did in the message so you cane easily see what changes were made later. EX: I built the "home" page" or "I fixed the tax calculation bug". Keep it short and sweet.

    git push

    Push your code to the main repo.
